# Netflix Data Analysis and Sentiment Analysis

## Overview
This project analyzes Netflix content data to extract meaningful insights related to content distribution, directors, actors, release trends, and sentiment of content descriptions.

## Features
- Content rating distribution analysis
- Top 5 directors analysis
- Top 5 actors analysis
- Year-wise content growth analysis
- Sentiment analysis of descriptions

## Tech Stack
- Python
- Pandas
- NumPy
- Plotly
- TextBlob
- Google Colab

## Dataset
Netflix Titles Dataset  
Rows: 8807  
Columns: 12

## Workflow
1. Load dataset
2. Clean missing values
3. Perform EDA
4. Generate visualizations
5. Apply sentiment analysis
6. Extract business insights

## Key Insights
- TV-MA and TV-14 dominate Netflix content.
- Content production increased significantly after 2000.
- Netflix contains a large amount of mature audience content.
- Sentiment analysis helps understand description tone.
